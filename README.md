# BTC Up/Down Jackpot

Une application de pari sur le Bitcoin avec jackpot progressif, intégrée à Base App.

## Fonctionnalités

- Pari Up/Down sur le BTC toutes les 24h
- Jackpot progressif en cas de flat
- Historique des paris
- Classement des joueurs
- Intégration Coinbase Wallet

## Stack Technique

- Frontend: React + Tailwind CSS
- Backend: Supabase
- Blockchain: Base L2 + Smart Contracts
- Oracle: Pyth Network

## Installation

```bash
npm install
npm start
```

## Déploiement

1. Frontend: Vercel
2. Backend: Supabase
3. Smart Contracts: Base L2

## Démarrage Rapide

1. Clonez le repo
2. `npm install`
3. `npm start`
4. Configurez Supabase
5. Déployez les smart contracts
